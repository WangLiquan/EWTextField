# EWTextField
<h3>Swift.定制textField</h3>

# 实现效果:

点击进入输入状态时placeholder变小上移,退出输入状态时若是textfield为空则恢复原状.

密码类型有明密文转换按钮.

使用extension实现通过修改属性控制最大输入数,默认15.
