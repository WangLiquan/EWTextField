# EWTextField
[![996.icu](https://img.shields.io/badge/link-996.icu-red.svg)](https://996.icu)

Swift.定制textField

# 实现效果:

点击进入输入状态时placeholder变小上移,退出输入状态时若是textfield为空则恢复原状.

密码类型有明密文转换按钮.

使用extension实现通过修改属性控制最大输入数,默认15.
